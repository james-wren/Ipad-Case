1. Attach case to screen when clossed.

3. Allow cover to attach to back of case and prop up for keyboard ussage
4. Activate the wake/sleep sensors in the ipad.

Once I finished the intial magnet model I created drawings and printed them so I could compare them with eachother and make sure the magnets lined up correct while also checking that the case cutouts for the buttons and camera were correct.
